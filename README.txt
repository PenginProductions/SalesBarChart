This C++ project was created using Microsoft Visual Studio Community Edition. This project was a small case study during a c++ course.

 2 files are contained:

	salesBarChart.cpp => contains contains main method.

	salesBarChart.exe => used to run program.
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
To Run:

	1) Launch salesBarChart.exe
__________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________________
Tips:

The idea of this program was to be able to enter raw sales data for multiple "stores"
and get an overview "bar graph" as the output. 

The output records an '*' for each 100 units per store; avoid to large of numbers as output is in command prompt.
